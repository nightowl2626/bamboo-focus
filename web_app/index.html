Setup

Connect the Pi

Bamboo Focus has two pieces: this app receives events and decides when to nudge; the Pi watches posture and objects locally, then sends only those results back here. No camera video is sent to the app.

Typical setup

1. Start Bamboo Focus here. 2. Calibrate from the main screen. 3. Start the Pi script below. 4. Start a focus session.

These three fields only build the SSH helper command. If you prefer to open a terminal on the Pi yourself, you can ignore the SSH command and use the manual Pi command below.

This is the address the Pi sends data to. In cloud mode, it should be your Alibaba URL. In local mode, it should be your laptop app URL.

Use this when your laptop can SSH into the Pi. It connects to the Pi, changes into the project folder, and starts camera monitoring with the right app address and token.

Use this if you are already connected to the Pi through Workbench, SSH, or a Pi terminal. Optional debug view: add --debug-stream, then open http://PI_IP:8765/.

This uses the laptop webcam instead of the Pi, which is useful for judging or demos when no Raspberry Pi is available. With debug enabled, open http://127.0.0.1:8766/.

Try It Locally

To make the most of Bamboo Focus, clone the GitHub repo on your laptop and run the local app. That gives you the web interface, Qwen nudges, calibration flow, and Pi or webcam demo scripts from one project.
